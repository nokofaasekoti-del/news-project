Insight Today
Project Description
Insight Today is a simple news website developed using HTML and CSS.
It provides users with the latest news, categories, multimedia content, and a contact page.
Pages Included
Home Page (index.html)
News Page (news.html)
Media Page (media.html)
Articles Page (article.html)
Contact Page (contact.html)
Features
Easy navigation menu
News categories
Embedded video and images
Image captions
Contact form
Responsive design for mobile devices
Technologies Used
HTML
CSS
How to Run the Project
Download the project folder
Open the folder
Click on index.html
The website will open in your browser
